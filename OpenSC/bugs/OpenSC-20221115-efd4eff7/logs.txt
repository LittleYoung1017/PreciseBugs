    Fix memory leak when freeing filesystem
    
    When freeing blobs with pgp_iterate_blobs(), blobs in the tree
    with a height higher than the given level will not be released.
    Since blobs should form a tree without cycles, counting levels
    can be omitted.
    
    Thanks OSS-Fuzz
    https://bugs.chromium.org/p/oss-fuzz/issues/detail?id=52671
